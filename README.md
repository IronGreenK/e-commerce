## Pre requisitos

- Node.js v16 (como mínimo)
- npm v8 (como mínimo)
- Un archivo `.env` que luzca como el archivo `.env.example`.

## Instalación

Para instalar las dependencias de este proyecto, corre en tu terminal:

```bash
yarn i
```

## Uso

Para correr el servidor en modo de desarrollo, utiliza:
```bash
yarn express
```
